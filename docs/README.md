# docs/ — Documents historiques

Ce dossier contient les documents du scoping initial du projet (profil utilisateur, reponses aux questions d'approfondissement, vision architecture).

## Statut

**Archive.** Ces fichiers ont servi de base pour construire les specs et la table `memory` Supabase. Les donnees live (profil, preferences, situation) sont maintenant dans la base de donnees.

## Contenu

- `profil/` — Profil professionnel, situation, problemes, personnalite
- `reponses/` — Reponses detaillees aux blocs A-I (routine, outils, finances, equipe, contenu, etc.)
- `architecture/` — Vision initiale de l'assistant IA, questions d'approfondissement

## Source de verite

- Pour les specs systeme : `specs/`
- Pour les donnees utilisateur live : table `memory` dans Supabase
- Pour le contenu formation : `learning-knowledge/` (curriculum, recherches, guides, modules)
