# Визуалы — Сессия 2: Зал и кухня

> Детальное описание каждого визуала для Сессии 2.
> Генерация: `npx tsx scripts/generate-visuels/index.ts`

---

## Общие правила

- **Минимализм** : 3-4 цвета, без визуального шума
- **Тёмный фон** `#1a1a2e` + яркие акценты
- **Шрифт** : Inter, sans-serif (жирный для заголовков)
- **Язык** : русский, технические термины в скобках на английском

### Цветовая палитра

| Роль | Hex | Применение |
|------|-----|------------|
| Клиент / Зал | `#4ECDC4` | Синий/бирюзовый |
| Сервер / Кухня | `#FF6B35` | Оранжевый |
| API / Официант | `#FFD166` | Жёлтый |
| БД / Холодильник | `#A855F7` | Фиолетовый |
| Фон | `#1a1a2e` | Тёмный |
| Текст | `#F5F5F5` | Белый |

### Форматы
- Слайды: **1920x1080** (16:9)
- Шпаргалка: **1080x1080** (квадрат для Discord)
