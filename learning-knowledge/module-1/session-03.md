## Философия С3

**С3 — переход от наблюдателя к создателю.** Студент впервые работает с кодом на своём компьютере, а не через платформу. С2 продвинулась быстрее, чем планировалось: вся архитектура (Клиент/API/Сервер/БД) усвоена и проверена квизом — результаты хорошие. Значит, пора действовать: установить среду, привыкнуть к IDE, создать первые файлы.

**Центральная идея** : вы больше не просите платформу сделать за вас — вы управляете ИИ в своём собственном пространстве.

**Фокус С3** : не строить сайт, а **привыкнуть к интерфейсу IDE** — папки, файлы, навигация, сохранение.

## ЧАСТЬ 3 — IDE: ваш рабочий стол (25 мин)

### Что такое IDE

IDE — это **редактор текста для кода**. Как Word, но для разработчиков:

- **Слева** — все папки и файлы вашего проекта
- **По центру** — содержимое открытого файла
- **Сверху** — вкладки открытых файлов (как вкладки в браузере)
- **Справа** — чат с ИИ

> "Проект состоит из множества файлов и папок. IDE делает управление всем этим удобным."

---

### Установка Antigravity (демо на экране)

1. Скачайте с **antigravity.google/download**
2. Установите (Mac: перетащите в Applications / Windows: запустите установщик)
3. Откройте Antigravity

**Первый запуск :**
- Choix d'import → **"Начать с нуля"** (fresh start)
- Theme → выбрать (без разницы)
- Автономия агента → **"Review-driven Development"** (рекомендуется)
- Авторизация → **"Sign in with Google"** → выбрать свой Gmail

---

### Экскурсия по интерфейсу (демо на экране)

**Показать по шагам :**

1. **Открыть папку** : Файл → Открыть папку → создайте папку "mon-site" на рабочем столе → откройте её
2. **Панель слева** : здесь отображаются все файлы и папки проекта
3. **Создать файл** : правый клик в панели слева → Новый файл → назовите "test.txt"
4. **Редактировать** : кликните на файл → напишите что-нибудь в центре экрана
5. **Вкладки** : откройте несколько файлов → они появляются сверху, как в браузере
6. **Создать папку** : правый клик → Новая папка → назовите "dossier-1"

### Сохранение файлов

В IDE рядом с названием файла на вкладке есть **точка** (●) — файл изменён, но не сохранён.

- **Mac** : `Cmd+S` для сохранения
- **Windows** : `Ctrl+S` для сохранения

После сохранения точка исчезает.

> "Если вы не сохраните файл, ваши изменения пропадут. Ctrl+S — ваш лучший друг."

---

### Навигация — ключевой навык

**Упражнение в прямом эфире (все делают одновременно) :**

1. Создайте в "mon-site" три папки : `page-1`, `page-2`, `page-3`
2. В каждой папке создайте файл `index.txt`
3. Напишите в каждом файле что-нибудь разное (имя, город, хобби)
4. Переключайтесь между файлами через вкладки и через панель слева
5. Сохраняйте каждый файл (`Cmd+S` / `Ctrl+S`)

> "Это упражнение может показаться простым, но c'est la base. Если вы не умеете быстро переключаться между файлами и папками, вы будете постоянно теряться."

## ЧАСТЬ 4 — ИИ + HTML (15 мин)

### Подключение ИИ

В Antigravity, ИИ **уже подключен**. Чат доступен справа или через `Cmd+L` (Mac) / `Ctrl+L` (Windows).

> Для тех, кто на VS Code + Codex : установите расширение "Codex" (editeur: OpenAI, id: `openai.chatgpt`), подключитесь с вашим compte ChatGPT+. **Rappel : support Windows experimental.**

### Что такое HTML — за 2 минуты

3 факта :

1. **Ваш сайт — на вашем компьютере.** Он ещё не в интернете, работает только у вас.
2. **Сайт работает в браузере** — Chrome, Safari и т.д.
3. **Браузер понимает HTML.** Это его "язык". Файл `.html` = страница сайта.

> "Для сегодняшнего дня запомните одно: HTML — это файлы, которые открываются в браузере и показывают содержимое сайта."

### Демо: первая страница

1. В чате ИИ, напишите :
   > "Создай мне файл index.html — простую страницу с моим именем по центру экрана"
2. ИИ создаёт файл — **пока не нужно понимать код**, просто наблюдайте
3. Найдите файл `index.html` в панели слева
4. Откройте его в браузере (правый клик → Open with → Chrome, или перетащите файл в браузер)
5. Страница отображается!

### Как обновить страницу после изменений

Попросите ИИ изменить что-нибудь, потом обновите в браузере:
- **Кнопка обновления** (стрелка) в навигаторе
- Или **кликните на URL** в адресной строке и нажмите **Enter**

> Не нужно закрывать и заново открывать файл.

### Не бойтесь сообщений

Во время работы появляются сообщения: "Подтверждаете?", "Разрешить?" — **не бойтесь**, можно подтверждать. Если не понимаете — спросите у ИИ.
