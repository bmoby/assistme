TITLE: Настройка среды разработки: IDE + ИИ + первый код

================================================================
BLOCK 1 — IDE и рабочее пространство
================================================================

[QCM]
Q1. Что такое IDE?

    A) Социальная сеть для программистов
    B) Редактор текста для кода — как Word, но для разработчиков
    C) Браузер для открытия сайтов
    D) Антивирусная программа

ANSWER: B
EXPLANATION: IDE — это среда разработки, редактор текста для кода. Там вы создаёте файлы, папки и управляете проектом.

---

[QCM]
Q2. Где в IDE отображаются все папки и файлы вашего проекта?

    A) По центру экрана
    B) Сверху, во вкладках
    C) В левой панели
    D) Справа, в чате с ИИ

ANSWER: C
EXPLANATION: Левая панель — это файловый менеджер проекта. Центр — содержимое файла, сверху — вкладки, справа — чат с ИИ.

---

[VF]
Q3. Если вы не сохраните файл в IDE (Ctrl+S / Cmd+S), ваши изменения всё равно сохранятся автоматически.

ANSWER: false
EXPLANATION: Несохранённый файл помечается точкой (●) на вкладке. Без Ctrl+S / Cmd+S изменения могут быть потеряны.

---

[OPEN]
Q4. Перечислите 4 основные зоны интерфейса IDE и что находится в каждой из них.

ANSWER: Слева — панель с файлами и папками проекта; по центру — содержимое открытого файла; сверху — вкладки открытых файлов; справа — чат с ИИ
EXPLANATION: Знание интерфейса — основа эффективной работы в IDE.

---

[QCM]
Q5. Почему Antigravity рекомендуется для начинающих?

    A) Он самый дорогой, значит самый лучший
    B) Он бесплатный, ИИ уже встроен, работает на всех системах
    C) Он работает только на Mac
    D) Он не требует интернета

ANSWER: B
EXPLANATION: Antigravity бесплатный, IDE и ИИ уже вместе, работает на Mac, Windows и Linux — идеальный старт без лишних шагов.

---

================================================================
BLOCK 3 — HTML и браузер
================================================================

[QCM]
Q6. Что такое HTML-файл?

    A) Картинка, которую можно открыть в Paint
    B) Файл, который браузер понимает и отображает как страницу сайта
    C) Программа, которая запускается на сервере
    D) Архив с несколькими файлами внутри

ANSWER: B
EXPLANATION: HTML — это язык, который браузер (Chrome, Safari и т.д.) читает и показывает как веб-страницу.

---

[VF]
Q7. Сайт, который вы создаёте на своём компьютере, сразу доступен в интернете для всех.

ANSWER: false
EXPLANATION: Пока сайт на вашем компьютере — он работает только у вас. Чтобы его увидели другие, его нужно разместить на сервере.

BLOCK 4 — Практика и навигация
================================================================

---

[QCM]
Q08. Какая комбинация клавиш сохраняет файл в IDE?

    A) Ctrl+C (Mac: Cmd+C)
    B) Ctrl+Z (Mac: Cmd+Z)
    C) Ctrl+S (Mac: Cmd+S)
    D) Ctrl+V (Mac: Cmd+V)

ANSWER: C
EXPLANATION: Ctrl+S (или Cmd+S на Mac) — сохранение файла. Это самая важная комбинация клавиш для разработчика.

---

[VF]
Q09. Чтобы попросить ИИ создать HTML-страницу, нужно сначала выучить код HTML.

ANSWER: false
EXPLANATION: Не нужно знать код — достаточно описать ИИ что вы хотите, и он создаст файл за вас. Понимание кода придёт постепенно.

---

[OPEN]
Q10. В чём разница между созданием страницы через платформу V0 (как на прошлой неделе) и через IDE на своём компьютере?

ANSWER: В IDE нет лимитов на изменения; полный контроль над файлами; ИИ работает прямо в рабочем пространстве; файлы хранятся на вашем компьютере
EXPLANATION: IDE даёт свободу и контроль, которых нет на онлайн-платформах с их ограничениями.
