# Visuels Session 1 — Briefs pour generation

> 6 visuels a creer. Le reste se dit a l'oral ou se montre en partage d'ecran.

---

## V1 — Slide titre

**Format** : Slide 16:9
**Texte** :
```
PILOTE NEURO — СЕССИЯ 2
[дата]

Добро пожаловать
```
**Style** : Sobre, fond sombre, typo bold. Logo si dispo.

---

## V2 — Journey Map (le visuel le plus important)

**Format** : Slide 16:9 ou infographie horizontale
**Contenu** :

Timeline horizontale, 6 blocs :

```
НЕД 1-2          НЕД 3-4          НЕД 5-6          НЕД 7-8          НЕД 9-10         НЕД 11-12
ОТКРЫТЬ           МЕТОД             АРСЕНАЛ ИИ       СТРОИТЬ           ПРО               ЗАПУСК
"Понимаю           "У меня есть      "У меня есть     "Проект            "Умею работать    "Проект онлайн,
цифровой мир"      метод"            все инструменты"  двигается"         как профи"        протестирован"
```

En dessous : courbe de motivation qui monte (sem 1-2), DESCEND (sem 4-5), remonte.
Annotation sur la vallee : **"Долина отчаяния — 100% студентов Сессии 1 через неё прошли. Это НОРМАЛЬНО."**

**Couleurs** : degrade vert clair → bleu fonce de gauche a droite. Vallee en rouge/orange.

---

## V3 — Regles du jeu

**Format** : Slide 16:9, deux zones
**Contenu** :

**Zone gauche — Rythme hebdo** :
```
📹 Видео до сессии (10-15 мин)
🔴 Лайв (2ч) — среда + воскресенье
📝 Задание между сессиями
⏰ Дедлайн — пятница 20:00
```

**Zone droite — Deadlines** :

```
✅ В срок (пт 20:00)    → Личный разбор от Магомеда
⚠️ С опозданием (вт 20:00) → Автопроверка
❌ Не сдал              → Ничего
```

**En bas** : "Дедлайны помогают ВАМ, а не наказывают."

**Format** : Slide 16:9 ou doc
**Contenu** : A remplir le jour J

ПОД 1 — Ментор: [Имя]     ПОД 2 — Ментор: [Имя]
• ...                        • ...
• ...                        • ...
• ...                        • ...

ПОД 3 — Ментор: [Имя]     ПОД 4 — Ментор: [Имя]

7-8 pods, 3-4 noms, mentor en gras. Equilibre par niveau tech.

---

## V5 — Quick Win (les 4 etapes)

**Format** : Slide 16:9, affiche pendant les 20 min de deploy
**Contenu** :

```
ТВОЙ ПЕРВЫЙ САЙТ ЗА 20 МИНУТ

1️⃣  v0.dev → "Sign in with Google"

2️⃣  Напиши промпт:
    "Создай страницу для [имя], [деятельность].
     Стиль современный, цвета [твой выбор]."

3️⃣  Нажми Deploy → скопируй URL

4️⃣  Вставь URL в Discord #победы

Застрял? → Пиши в #под-[номер]
```

**Style** : Gros chiffres, lisible meme de loin. Captures d'ecran v0 a cote de chaque etape si possible.

---

## V6 — Guide Quick Win detaille (post-session, PDF/post Discord)

**Format** : PDF 1-2 pages ou post long dans #задания
**Ce n'est PAS un slide** — c'est le document poste APRES le live.

```
ЗАДАНИЕ НЕДЕЛЯ 1 — Твой первый сайт

ЭТАП 1 — Аккаунт v0 (2 мин)
[скриншот: страница v0.dev]
→ Зайди на v0.dev → "Sign in" → Google → Готово

ЭТАП 2 — Создай страницу (15 мин)
[скриншот: поле для промпта]
→ Напиши промпт. Примеры:
  • "Landing page для фитнес-тренера, тёмный минимальный стиль"
  • "Портфолио фотографа с галереей"
  • "Страница кондитера с меню и контактами"
→ Не нравится? Напиши, что изменить.

ЭТАП 3 — Задеплой (2 мин)
[скриншот: кнопка Deploy]
→ Нажми "Deploy" или "Publish"
→ Vercel просит аккаунт? → "Continue with Google"
→ Скопируй URL (xxx.vercel.app)

ЭТАП 4 — Поделись (2 мин)
→ URL в Discord #победы
→ Отправь URL 1 человеку (семья, друг)

ЗАСТРЯЛ?
→ Скриншот проблемы в #под-[номер] или #faq
→ Ментор поможет за 24ч

ДЕДЛАЙН: Пятница 20:00

БОНУС: Улучши страницу — добавь разделы "Обо мне", "Проекты", "Контакт".
```

## Ce qui ne necessite PAS de visuel

| Element | Pourquoi |
|---------|----------|
| Comment poser une question | Se dit a l'oral en 30 sec + le template est dans les regles Discord |
| Les canaux Discord | Partage d'ecran en live, plus authentique qu'un slide |
| Le bot | Demo live en 2 min, pas besoin de slide |
| Les 6 modules en detail | Journey Map (V2) suffit, le detail est oral |
