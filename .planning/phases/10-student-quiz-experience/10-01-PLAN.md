    Import from discord.js: `EmbedBuilder`, `ActionRowBuilder`, `ButtonBuilder`, `ButtonStyle`.
    Import types from `@assistme/core`: `QuizQuestion`, `StudentQuizAnswer`.

    Export these 5 pure functions:

    a) `buildQuestionEmbed(q: QuizQuestion, questionNumber: number, totalQuestions: number): EmbedBuilder`
       - Title: `Вопрос ${questionNumber}/${totalQuestions}`
       - Description: `q.question_text`
       - Color: `0x5865F2` (Discord blurple)
       - Footer text: map type -> `mcq` = "Выбор ответа", `true_false` = "Правда/Ложь", `open` = "Открытый вопрос"

    b) `buildMcqRow(sessionId: string, choices: Record<string, string>): ActionRowBuilder<ButtonBuilder>`
       - Iterate `Object.entries(choices)` sorted by key (A, B, C, D order)
       - Each button: customId = `quiz_answer_${sessionId}_${key}`, label = `${key}: ${label}`, style = `ButtonStyle.Primary`
       - Cast `choices` defensively: `const safeChoices = choices as Record<string, string>`

    c) `buildTrueFalseRow(sessionId: string): ActionRowBuilder<ButtonBuilder>`
       - Two buttons: customId `quiz_answer_${sessionId}_true` label "Правда" style Success, customId `quiz_answer_${sessionId}_false` label "Ложь" style Danger

    d) `buildOpenQuestionEmbed(q: QuizQuestion, questionNumber: number, totalQuestions: number): EmbedBuilder`
       - Same as buildQuestionEmbed but add a field: name = "Ответ", value = "Напишите ваш ответ в чат", inline = false

    e) `buildFeedbackMessage(answers: StudentQuizAnswer[], questions: QuizQuestion[], score: number): string`
       - Per D-21: header = `**Результат: ${correct}/${total} (${Math.round(score)}%)**\n\n`
       - Per D-22: For each question, find matching answer. Correct answers: `✅ **Q${q.question_number}:** ${ans.student_answer}`. Incorrect answers: `❌ **Q${q.question_number}:** ${ans.student_answer}\n> Правильно: ${q.correct_answer}` + if explanation exists and length > 0: `\n> ${explanation}` (truncate explanation to 120 chars if longer, append "...")
       - Skipped questions (no answer): `⬜ Вопрос ${q.question_number}: пропущен`
       - Per D-23: if total message > 2000 chars, truncate body at last `\n` before limit, append "..."
       - `const MAX_MSG_LEN = 2000; const EXPLANATION_HARD_CUT = 120;`

    ```typescript
    export async function evaluateOpenAnswer(question: QuizQuestion, studentAnswer: string): Promise<EvalResult>
    ```

    - Per D-18: System prompt (Russian): `"Ты — строгий, но справедливый преподаватель. Оценивай ответы студентов семантически: принимай синонимы, перефразировки и частичные совпадения если основная идея верна. Только полностью неверные или нерелевантные ответы считаются неправильными. Отвечай JSON: {\"isCorrect\": boolean, \"reasoning\": string}"`
    - Prompt: include `question.question_text`, `question.correct_answer`, and `studentAnswer`
    - Parse response with `JSON.parse(raw)` in try/catch
    - Fallback on parse failure: `studentAnswer.toLowerCase().includes(question.correct_answer.toLowerCase())` with reasoning `'parsing fallback'`

    **5. Create `packages/bot-discord-quiz/src/utils/quiz-flow.ts`:**

    Import from `@assistme/core`: `updateQuizSession`, `getAnswersBySession`, `logger`, types `StudentQuizSession`, `QuizQuestion`.
    Import from discord.js: `TextBasedChannel`.
    Import from `./quiz-messages.js`: `buildQuestionEmbed`, `buildMcqRow`, `buildTrueFalseRow`, `buildOpenQuestionEmbed`, `buildFeedbackMessage`.

    Export two functions:

    a) `sendQuestion(dmChannel: TextBasedChannel, session: StudentQuizSession, question: QuizQuestion, totalQuestions: number): Promise<void>`
       - Build embed via `buildQuestionEmbed(question, session.current_question + 1, totalQuestions)`
       - If `question.type === 'mcq'`: build row via `buildMcqRow(session.id, question.choices as Record<string, string>)`, send embed + components
       - If `question.type === 'true_false'`: build row via `buildTrueFalseRow(session.id)`, send embed + components
       - If `question.type === 'open'`: use `buildOpenQuestionEmbed`, send embed only (no buttons — student types text per D-13)
       - Return the `awaitingOpenAnswer` state is NOT managed here — that belongs in the DM handler. This function only sends.

    b) `advanceOrComplete(userId: string, session: StudentQuizSession, questions: QuizQuestion[], dmChannel: TextBasedChannel): Promise<void>`
       - Calculate `nextIndex = session.current_question + 1`
       - If `nextIndex >= questions.length`: quiz complete
         - Fetch answers via `getAnswersBySession(session.id)`
         - Calculate `correctCount = answers.filter(a => a.is_correct).length`
         - Calculate `score = questions.length > 0 ? (correctCount / questions.length) * 100 : 0`
         - Update session: `updateQuizSession(session.id, { status: 'completed', score, completed_at: new Date().toISOString() })`
         - Build feedback: `buildFeedbackMessage(answers, questions, score)`
         - Send feedback to dmChannel
       - Else: advance
         - Update session: `updateQuizSession(session.id, { current_question: nextIndex })`
         - Get `nextQ = questions[nextIndex]`
         - Call `sendQuestion(dmChannel, updatedSession, nextQ, questions.length)`
       - Return the updated session (for the DM handler to set awaitingOpenAnswer if needed). Change return type to `Promise<StudentQuizSession | null>` — returns the updated session on advance, null on complete.
