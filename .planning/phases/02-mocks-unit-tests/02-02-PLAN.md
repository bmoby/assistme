**Mock client factory** (same pattern referenced in research):
Create a `makeClient()` helper that returns a minimal Client-like object with `on()` that stores handlers in a Map and a custom `__emit(event, ...args)` method to invoke them. This avoids importing the real Client constructor.

```typescript
function makeClient() {
  const handlers = new Map<string, Function>();
  return {
    on: vi.fn((event: string, handler: Function) => { handlers.set(event, handler); }),
    __emit: async (event: string, ...args: unknown[]) => {
      const handler = handlers.get(event);
      if (handler) await handler(...args);
    },
  };
}
```
