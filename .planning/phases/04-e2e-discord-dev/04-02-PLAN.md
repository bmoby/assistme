- Result contains `{ text: string, submissionId?: string }`
- Sends reply via `message.reply()` or `sendLongMessage()`
- File attachments: downloads `attachment.url` via `fetch()`, validates MIME + size
- Max file size: 25 MB

- Triggered by `messageCreate` when `channel.name === CHANNELS.faq` (i.e., 'faq')
- Checks `message.author.bot` → return if bot
- Checks `isStudent(message)` → return if not student/admin/mentor
- Question length must be >= 5 chars
- Calls `answerFaqQuestion({ question, existingFaq })` from @assistme/core
- Sends reply in same channel
