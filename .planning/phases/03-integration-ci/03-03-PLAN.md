Create `packages/core/src/ai/formation/dm-agent.integration.test.ts` that tests the DM Agent against a real Supabase instance while keeping the Claude API mocked.

**Pattern:** The DM Agent internally calls `getStudentByDiscordId()` and other DB functions via `getSupabase()`. Since the integration test project's env points to real local Supabase (http://127.0.0.1:54321), these DB calls hit the real database. Claude API is mocked via `vi.mock('@anthropic-ai/sdk')` to return deterministic responses.

**Read dm-agent.ts first** to understand:
1. The exact export name and signature of the main agent function (likely `runDmAgent` or similar)
2. Which DB functions it calls (getStudentByDiscordId, searchFormationKnowledge, etc.)
3. What parameters it expects (discord_id, message content, etc.)
4. What it returns (response text, actions, etc.)

**Test structure:**
```typescript
import { vi, describe, it, expect, beforeAll, afterAll, beforeEach } from 'vitest';
import { createTestClient, createTestRunId, cleanupTestData } from '../../../../test/integration-helpers.js';

// Mock Claude API — must be before any import that loads @anthropic-ai/sdk
const mockCreate = vi.fn();
vi.mock('@anthropic-ai/sdk', () => ({
  default: vi.fn().mockImplementation(() => ({
    messages: { create: mockCreate },
  })),
}));

// Mock embeddings — dm-agent may call getEmbedding which needs embedding server
vi.mock('../../ai/embeddings.js', () => ({
  getEmbedding: vi.fn().mockResolvedValue(null),
}));

// Now import the agent (after mocks are set up)
// Adjust import path based on actual dm-agent.ts exports
import { runDmAgent } from './dm-agent.js'; // or whatever the actual export is

const TEST_RUN_ID = createTestRunId();
const adminDb = createTestClient();
const TEST_DISCORD_ID = `discord-intg-${TEST_RUN_ID}`;

beforeAll(async () => {
  // Seed a real student in the database
  const { error } = await adminDb.from('students').insert({
    name: `${TEST_RUN_ID}-IntegStudent`,
    discord_id: TEST_DISCORD_ID,
    session: 2,
    status: 'active',
    payment_status: 'paid',
  });
  if (error) throw new Error(`Failed to seed student: ${error.message}`);
});

afterAll(async () => {
  await cleanupTestData(adminDb, 'students', 'name', TEST_RUN_ID);
});

beforeEach(() => {
  mockCreate.mockReset();
});
```
